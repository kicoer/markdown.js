# markdown.js

a simple Markdown parser for JavaScript.

## syntax

* \`code\`
* \*italicize text\*
* \*\*bold text\*\*
* \[link text\]\(link address\)
* \!\[alt text\]\(image src\)
* \#\# heading
* \* unordered list
* 1\. ordered list
* \escape
* \> blockquotes
* code
```
​```php
<?php
    echo "mdzz";
​```
```
* table
```
| left align | right align | center |
| :------| ------: | :------: |
| AND | 0 | 1 |
| 0 | 0 | 0 |
| 1 | 0 | 1 |
```

## usage

```js
html_str = markdown(markdown_str);
```
