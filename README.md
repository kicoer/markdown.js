# markdown.js

a simple Markdown parser for JavaScript.

## Syntax:

* \`code\`
* \*italicize text\*
* \*\*bold text\*\*
* \[link\]\(link address\)
* \!\[alt text\]\(image\)
* \#\# heading
* \* unordered list
* 1\. ordered list
* \escape
* \> blockquotes
